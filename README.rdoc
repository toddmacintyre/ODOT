This is a TO DO list web application built using Ruby on Rails 4.

User can Mark TO DO items as complete as well as add new lists and items, edit, save, and delete the items.

Built using these versions:
ruby 2.0.0p353 (2013-11-22 revision 43784) [i686-linux]
Rails 4.0.1
rspec 2.99.2

Please check the Gemfile for specific dependencies.
